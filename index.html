Revealo（リビーロ）とは？新感覚のパズル体験

Revealo（リビーロ）は、従来の「イラストロジック（ノノグラム、ピクロス）」や「ナンプレ（数独）」の面白さを融合させ、さらに色彩の楽しさを加えたウェブブラウザ向けの完全無料パズルゲームです。

プレイヤーに与えられるのは、行と列に並んだ「数字」と「カラーチップ」のヒントだけ。これらの条件を論理的に整理し、空白のマス目に正しい色を塗っていくことで、隠されたイラストを少しずつ浮かび上がらせていきます。イラストが何であるかを予想する推理要素もあり、最初から最後まで飽きずに頭の体操を楽しむことができます。

パズルの基本ルール

Revealoのルールは非常にシンプルで、かつ奥深いものです：

  • ヒントチップの数字： 各行・各列に並ぶ数値は、その列に「同じ色のマスが連続して塗られる数」を表しています。
  • 色の順序： ヒントに並んでいる色の順番通りに、上から下（または左から右）へマスを塗る必要があります。
  • スキップ（空白マス）： 色のブロックの間には、1マス以上の空白（または異なる色）が入ります。ただし、具体的な空白の数はヒントには書かれていません。論理的に絞り込んでいく必要があります。

他のペンシルパズル（ピクロスやナンプレ）との違い

一般的なピクロスでは「塗るか・塗らないか」の2択ですが、Revealoはカラーイラストに対応しています。カラーピクロスと異なり、Revealoのヒントは色と連続数のペアに加えて「一部のヒントが隠されている（難易度Normal以上）」など、より推理力を試されるギミックが満載です。ナンプレのような徹底した論理的消去法と、ノノグラムのビジュアル的な楽しさの両方を同時に味わうことができます。

攻略のヒントと初心者ガイド

1. 確定する大きい数字から攻める

パズルを解き始める際は、行または列のサイズ（例：10x10なら「10」に近い数字のヒント）に近い大きな数字を探します。確実に塗れるマスを先にマークすることで、周辺のヒントも自ずと確定しやすくなります。

2. 消去法（×マーク）を活用する

絶対に色が塗られないと分かったマスには、パレットの「消す（×）」ツールを使って空白を確定させます。視覚的に候補を絞り込むことが、クリアタイムを縮める最大の秘訣です。

3. 「何の絵？」推理機能を使う

盤面がある程度埋まってくると、「何の絵？」ボタンからイラストの正体を予想することができます。正解を選択すると、未確定のマスにヒントが表示されるクリアボーナスが得られ、難しい問題もスッキリ解決できます。

4. 難易度と盤面サイズを選ぶ

初心者の方は8x8のEasyパズルから開始し、慣れてきたら10x10、12x12、さらに難解な15x15のExpertレベルにステップアップしていきましょう。論理パズルとしてのやりごたえがどんどん増していきます。

About Revealo - Picture Logic Coloring Puzzle

Revealo is a free web-based logic puzzle game. By combining visual nonograms (picross) and Sudoku reasoning, it delivers an innovative deduction experience using rich color clues and numerical patterns.

Gameplay Rules

Your goal is to fill the blank cells with the correct colors by reading the color blocks and numbers displayed next to the rows and columns. Each number represents how many consecutive cells of that color exist, and they appear in the specified sequence.

Play on Mobile as a PWA

Install the game onto your mobile home screen via Chrome or Safari. It works offline and offers full-screen premium visual performance just like native applications.
